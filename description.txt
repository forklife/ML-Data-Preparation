**Описание проекта**  
Проект для создания датасета прогнозирования цен XAUUSD на таймфрейме M15 с использованием данных MetaTrader5. Основной скрипт `dataset_builder.py` выполняет ETL: загрузка данных, трансформация, расчёт индикаторов (RSI, MACD, Bollinger Bands, Super Trend, ATR, ADX, CCI, историческая волатильность, свечные паттерны, candle_blue, candle_red и др.), валидация и сохранение датасета в `output-datasets-fortraining/dataset_XAUUSD_15.new`. Интерполяция устраняет NaN, валидация проверяет диапазоны индикаторов. Текущая версия: ~126 колонок (включая ~61 свечной паттерн, candle_blue, candle_red), ~12 секунд на расчёт индикаторов.

**Структура проекта**  
- `dataset_builder.py` (v1.0): Координирует ETL-процесс.
- `description.txt` : Описание проекта - необходимо поддерживать в актуальном состоянии по мере обновления проекта.
- `change.txt` : Описание всех изменений и обновлений - необходимо обновлять по мере любых обновлений изменений.
- `components/`:

  - `indicators_metrics_builder.py` (v0.26): Расчёт ~126 индикаторов.

  - `data_validation.py` (v0.11): Проверка типов, диапазонов, NaN.
  - `debug.py` (v0.3): Логирование.
  - `version_control.py` (v1.16): Отслеживание версий.

  - `mt5_api.py` (v1.3): Интерфейс MT5.
- `output-datasets-fortraining/`: Папка для датасетов.
